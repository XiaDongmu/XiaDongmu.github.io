Java中栈和堆的区别

发表于 2018-07-02 | 分类于 Java基础 |

在Java语言中，堆和栈都是内存中存放数据的地方。变量分为基本数据类型和引用类型，基本数据类型的变量（例如int、short、long、byte、float、double、boolean以及char等）以及对象的引用变量，其内存都分配在栈上，变量出了作用域就会自动释放，而引用类型的变量，其内存分配在堆上或者常量池（例如字符串常量和基本数据类型常量）中，需要通过new等方式创建。
