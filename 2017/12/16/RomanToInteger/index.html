Roman To Integer

Given a roman numeral, convert it to an integer.
Input is guaranteed to be within the range from 1 to 3999.

计数规则：

  1. 相同的数字连写，所表示的数等于这些数字相加得到的数，例如：III = 3
  2. 小的数字在大的数字右边，所表示的数等于这些数字相加得到的数，例如：VIII = 8
  3. 小的数字，限于（I、X和C）在大的数字左边，所表示的数等于大数减去小数所得的数，例如：IV = 4
  4. 正常使用时，连续的数字重复不得超过三次
  5. 在一个数的上面画横线，表示这个数扩大1000倍（本题只考虑3999以内的数，所以用不到这条规则）
  6. 其次，罗马数字转阿拉伯数字规则（仅限于3999以内）：

编程思路：

从前向后遍历罗马数字，如果某个数比前一个数小，则加上该数。反之，减去前一个数的两倍然后加上该数（我用java里的hashmap做一个小字典，然后进行前后字符的判断，最后写出的这个程序）

package com.eric.RomanToInteger;

import java.util.HashMap;
import java.util.Scanner;

public class RomanToInteger {
public static void main(String[] args) {
    HashMap<Character,Integer> hashMap=new HashMap<>();
    hashMap.put('I',1);
    hashMap.put('V',5);
    hashMap.put('X',10);
    hashMap.put('L',50);
    hashMap.put('C',100);
    hashMap.put('D',500);
    hashMap.put('M',1000);

    Scanner scanner=new Scanner(System.in);
    String string=scanner.nextLine();

    char[] c=string.toCharArray();
    int result=hashMap.get(c[0]);
    for (int i = 0; i <c.length-1 ; i++) {
        if (hashMap.get(c[i])>hashMap.get(c[i+1])){
            result=result+hashMap.get(c[i+1]);
        }else{
            result=result-hashMap.get(c[i])*2+hashMap.get(c[i+1]);

        }
    }

    System.out.println(result);
    }
}
