Reverse Integer (LeetCode)

题目:

Given a 32-bit signed integer, reverse digits of an integer.

Example1:

Input:       123
Output:      321

Example2:

Input:      -123
Output:     -321

Example3:

Input:        120
Output:        21

解法一：

用数学方法解决问题，原理是将求出整数的每一位，然后在加到一块儿，实现反转

**class Solution{
        int reverse(int x){
            int res=0;
            while(x!=0){
                res=res*10+x%10;
                x/=10;
        }
    return (res>Integer.MAX_VALUE||res<Integer.MIN_VALUE)?0:res；
    }
}

1. 使用数组

public static String strReverseWithArray(String string){
            if(string.length()==0||string==null) return string;
            int length=string.length;
            char[] array=string.toCharArray();
            for(int i=0;i<length;i++){
                array[i]=string.charAt(length-1-i);

        }
        return new String(array);
    }

2、使用栈

public static String reverseWithStack(String string){
            if(string==null||string.length()==0) return string;
            char[] c=string.toCharArray();
            Stack<Character> stack=new Stack<>();

            for(int i=0;i<c.length;i++){
                stack.push(c);                    
            }

            for(int i=0;i<c.length;i++){
                c[i]=stack.pop();

        }
    return new String(c);

    }
